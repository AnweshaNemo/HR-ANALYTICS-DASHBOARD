# HR Analytics Dashboard

Interactive HR Analytics Dashboard built using Power BI to analyze employee attrition, job satisfaction, salary trends, workforce distribution, and departmental performance.

## Tools Used
- Power BI
- Excel
- Power Query
- DAX

## Features
- Employee Attrition Analysis
- Department-wise Insights
- Salary Trend Analysis
- Interactive Visualizations
- KPI Cards and Filters
